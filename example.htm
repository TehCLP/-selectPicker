Example : $selectPicker

Options : Default 
                                
    <script type="text/javascript">
        $selectPicker.init({

            idDay: '#day1'
            , idMonth: '#month1'
            , idYear: '#year1' 

        });
    </script>
Options : TimeAfter
สำหรับ check เวลา หลังจากเวลาที่กำหนดให้ +1 วัน
เช่น check หลังจาก 10.00 น. บังคับให้เลือก วันถัดไป ทันที 
    <script type="text/javascript">
        $selectPicker.init({

            idDay: '#day3'
            , idMonth: '#month3'
            , idYear: '#year3' 

            // type : string => format 'hh.mm' or 'hh:mm'
            , timeAfter: '10.00'

        });
    </script>
Options : Holiday & TimeAfter

สำหรับ check วันที่ ช่วงหยุดยาว

option : holiday
- startDate : วันที่เริ่มหยุด
- endDate : ถึงวันที่
- toDate : ถ้าอยู่ในช่วงวันที่กำหนด บังคับให้เลือกได้ ตั้งแต่วันที่

option : timeAfter
- ในกรณี ใส่ timeAfter : จะ check เวลา หลังจากเวลาที่กำหนดให้ +1 วัน เฉพาะวันที่เริ่ม

เช่น ในตัวอย่าง
กำหนดถ้า วันที่ปัจจุบัน อยู่ในช่วงวันที่ 01/04/2016 - 10/04/2016 จะบังคับให้เลือกได้ ตั้งแต่วันที่ 12/04/2016 เป็นต้นไป 

    <script type="text/javascript">
        $selectPicker.init({

            idDay: '#day5'
            , idMonth: '#month5'
            , idYear: '#year5'

            // type : string => format 'hh.mm' or 'hh:mm'
            , timeAfter: '10.00' 

            , holiday: {

                // type : string => format : yyyy-mm-dd
                // type : Date

                startDate: '2016-04-01' // วันที่ เริ่มหยุด 
                , endDate: '2016-04-10' // ถึงวันที่
                , toDate: '2016-04-12'  // วันที่ กำหนดให้ เริ่ม
            }

        });
    </script>
Options : Language
* ในกรณีที่ต้องการใช้มากกว่า 1 ชุด สามารถ binding ได้ทีชุด 
    <script type="text/javascript">
        $selectPicker.init({

            idDay: '#day2'
            , idMonth: '#month2'
            , idYear: '#year2'
            , language: 'en' // 'en' or 'th'

        });
        $selectPicker.init({

            idDay: '#day2th'
            , idMonth: '#month2th'
            , idYear: '#year2th'
            , language: 'th' // 'en' or 'th'

        });
    </script>
Options : Min - Max
สำหรับ render range date ( วันที่เริ่มต้น และ วันที่สิ้นสุด )
เช่น ใช้ render select สำหรับเลือก วัน/เดือน/ปี เกิด ในช่วงอายุที่กำหนด 
    <script type="text/javascript">
        $selectPicker.init({

            idDay: '#day2'
            , idMonth: '#month2'
            , idYear: '#year2'
                                
            // type : string => format : yyyy-mm-dd
            // type : Date
            , minDate: '2014-10-01' 

            // type : string => format : yyyy-mm-dd
            // type : Date
            , maxDate: '2016-10-01'

        });
    </script>
